Featured Project: Bayesian Hierarchical Pitcher Aging & Decline Detection Project

This advanced hierarchical Bayesian framework uses R to model individual pitcher aging trajectories and detect performance decline, processing 10+ years of Statcast data to provide front offices with injury risk assessments and workload management insights.


Phase 1: Architected hierarchical Bayesian model to estimate age curves for velocity, movement, and command metrics across pitcher populations. Established decline patterns and individual risk assessments using 10+ years of Statcast data with uncertainty quantification through posterior distributions.


Phase 2: Implemented changepoint detection algorithms to identify precise decline onset timing with statistical confidence intervals. System distinguishes between natural aging, mechanical degradation, and injury-related performance changes


Phase 3: Developed real-time posterior updating system incorporating new performance data to continuously refine decline probability estimates. Created interactive Shiny Apps dashboard translating complex Bayesian posterior distributions into actionable insights for coaching staff and front office personnel.

xGB% Model

An R-based machine learning pipeline using XGBoost to predict xGB% on ball-flight characteristics only. The pipeline involves data preprocessing and filtering, feature engineering, model training with cross-validation, performance and accuracy evaluation (ROC-AUC calculations), feature importance and correlation plots analysis, and pitcher leaderboard table creation. I also developed a interactive Shiny App calculator for real-time metric calculation for pitchers on their pitch characteristics to predict expected groundball rates. The plots help determine which model features are significant in influencing a GB outcome, and the final leaderboard compares actual vs. expected GB rates and identifies over/under-performers relative to their pitch characteristics.

Deacon Financial Services

Engineered a comprehensive fraud detection system processing 900,000+ loan applications using Python and scikit-learn. Performed extensive data preprocessing including handling class imbalance through strategic undersampling, feature engineering, and exploratory data analysis to identify fraud patterns. Constructed and evaluated multiple classification models—including Logistic Regression, Weighted Logistic Regression, Random Forest, and XGBoost—achieving 88% fraud detection rate with optimized precision-recall tradeoffs. Implemented a risk-stratified 3-tier classification system (high/medium/low risk) using probability threshold optimization to minimize false negatives while maintaining operational efficiency. Delivered actionable insights through comprehensive feature importance analysis, ROC-AUC evaluation (0.92 AUC), and confusion matrix interpretation, enabling data-driven decision-making for loan approval workflows. Project demonstrates proficiency in handling severely imbalanced datasets, ensemble methods, hyperparameter tuning, and translating machine learning predictions into business logic for financial risk management.
